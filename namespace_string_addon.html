Function which splits a string according to a given splitter substring and returns the split string in a list.

Parameters
inputStringInput string to be split
splitterStringString to split the inputString by
isCombineBoolean to indicate if the user wants to split the string and combine multiple empty strings in the list.
isCombine == true, ("var,,,a", ",") --> list = {var}, {a};
(default) isCombine == false, ("var,,,a,", ",") --> list = {var}, {}, {}, {a}, {}
Returns
List of strings split by splitterString. If splitterString does not exist, returns a list with the inputString as the first and only item.
